## 🎮 游戏介绍

**《我的中餐厅》** 是一款中国风文字放置经营游戏，玩家经营一家中餐厅，雇佣厨师、升级菜品、装修餐厅，体验从小店到连锁餐厅的成长过程。

**特色：**
- 🤖 **AI 生成**：代码、文案、数值设计全部由 AI 完成
- 🏮 **中国风**：传统配色、楷体字体、中式元素
- 📱 **微信小游戏**：扫码即玩，无需下载
- 💰 **放置玩法**：离线也有收益，轻松不肝

## 🚀 快速开始

### 方式 1：微信开发者工具（推荐）

#### 第 1 步：下载工具

访问 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

下载并安装 **微信开发者工具**

---

#### 第 2 步：导入项目

1. 打开微信开发者工具
2. 登录微信开发者账号
3. 点击 **导入项目**
4. 选择项目目录：`minigame/`
5. 填写 AppID（测试可选"测试号"）
6. 点击 **导入**

---

#### 第 3 步：编译运行

1. 点击工具栏 **编译** 按钮
2. 在模拟器中查看游戏
3. 点击 **预览** 生成二维码
4. 微信扫码在手机上体验

---

### 方式 2：直接体验（需上传）

1. 注册微信小程序账号
2. 上传代码到微信后台
3. 设置为体验版
4. 添加体验者（10 人）
5. 分享二维码给体验者

## 🎯 游戏玩法

### 核心循环

```
雇佣厨师 → 接待客人 → 获得金钱 → 升级菜品/装修 → 吸引更多客人
```

### 系统说明

#### 1. 厨师系统

- 初始拥有 **王师傅**（川菜）
- 可雇佣更多厨师（粤菜、鲁菜、淮扬菜）
- 升级厨师提高效率

#### 2. 菜品系统

- 初始解锁 3 道菜品
- 可解锁更多菜品（鱼香肉丝、红烧肉等）
- 升级菜品提高单价

#### 3. 装修系统

- **桌椅**：增加接待桌数
- **装饰**：提升餐厅美观度
- **厨房**：提高出菜效率

#### 4. 成就系统

- 10 个成就等待解锁
- 达成成就获得金币奖励

## 🛠️ 自定义修改

### 修改 AppID

编辑 `project.config.json` 和 `app.json`：

```json
{
  "appid": "你的 AppID"
}
```

---

### 修改云开发环境

编辑 `app.js`：

```javascript
wx.cloud.init({
  env: 'your-env-id',  // 替换为你的云开发环境 ID
})
```

---

### 调整数值

编辑 `app.js` 中的 `globalData`：

```javascript
globalData: {
  money: 200,          // 初始资金
  chefs: [...],        // 厨师配置
  dishes: [...],       // 菜品配置
  decorations: {...},  // 装修配置
}
```

---

### 添加新菜品

在 `pages/index/index.js` 的 `unlockNewDish()` 函数中添加：

```javascript
{ id: 106, name: '新菜品', price: 30, level: 1, cost: 12, unlockCost: 400 }
```

### 添加新厨师

在 `pages/index/index.js` 的 `hireNewChef()` 函数中添加：

```javascript
{ id: Date.now(), name: '新厨师', level: 1, efficiency: 1.3, cost: 250, specialty: '菜系' }
```

## 📊 数据说明

### 收益公式

```
每秒收益 = 菜品收入 × 厨师加成 × 装修加成

菜品收入 = Σ(菜品价格 × 等级 × 0.5)
厨师加成 = 1 + Σ(厨师效率 × 等级 × 0.1)
装修加成 = 1 + (桌椅等级×0.2) + (装饰等级×0.15) + (厨房等级×0.15)
```

### 离线收益

```
离线收益 = 每秒收益 × 离线时间 × 0.8
```
